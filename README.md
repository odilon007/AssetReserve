# Projeto PW2: Carros Antigos

Este projeto foi desenvolvido como parte da disciplina **PW2 (Programação para Web 2)** e tem como objetivo apresentar um site interativo sobre **carros antigos**, utilizando **HTML**, **CSS** e **JavaScript**. O site exibe uma galeria de carros antigos, permite ao usuário ver detalhes sobre os modelos e interagir com o conteúdo de forma dinâmica.

## Funcionalidades

- **Galeria de Carros Antigos**: Exibição de diversos modelos de carros antigos com imagens e detalhes.
- **Modal de Detalhes**: Ao clicar em um carro, o usuário pode visualizar informações adicionais sobre o modelo, como ano, fabricante e características.
- **Linha do Tempo Interativa**: Uma linha do tempo mostrando a evolução dos carros ao longo das décadas.
- **Quiz de Carros Antigos**: Um quiz interativo para testar os conhecimentos dos usuários sobre carros antigos.
- **Estilo Responsivo**: O layout do site é responsivo, adaptando-se a diferentes tamanhos de tela (desktop, tablet e celular).

## Tecnologias Utilizadas

- **HTML**: Para estruturar o conteúdo da página.
- **CSS**: Para estilizar a página e criar um design moderno e responsivo.
- **JavaScript**: Para criar interatividade no site, como a exibição de detalhes dos carros e a implementação do quiz.
- **Bootstrap** (opcional): Para facilitar o design responsivo e a criação de componentes como botões e modais.
